# Industrial Foregoing: Simulation


## zh_cn
##### 添加了模拟自动刷怪机
- 用于将**生物捕捉工具**内的怪物数据存入模拟卡内

##### 添加了模拟生物粉碎机
- 用于通过存有**怪物数据**的模拟卡产出怪物资源
- 需要从**模拟自动刷怪机**获取模拟数据

##### 添加了模拟矿物镭射钻
- 用于通过存有**矿物数据**的模拟卡产出矿物资源
- 需要从**镭射钻基座**中获取模拟数据

##### 添加了模拟流体镭射钻
- 用于通过存有**流体数据**的模拟卡产出流体资源
- 需要从**流体镭射钻基座**中获取模拟数据
- 只能从**不需要生物**的流体镭射钻配方中获取流体数据

##### 与新生魔艺的联动
- **需要生物**的**流体镭射钻配方**现在可以用对应生物的**收容罐**代替(放在机器下方一格)
- **灵魂镭射基座**所需的监守者可以是**收容了监守者的收容罐**(放在机器下方一格)

## en_us
##### Added Simulated Mob Duplicator
- Used to store mob data from **Mob Imprisonment Tool** into simulated card

##### Added Simulated Mob Crusher
- Used to generate mob resources through a simulated card containing **mob data**
- Need to obtain simulation data from **Simulated Mob Duplicator**

##### Added Simulated Ore Laser Base
- Used to generate ore resources through simulated cards containing **ore data**
- Need to obtain simulation data from the **Ore Laser Base**

##### Added Simulated Fluid Laser Base
- Used to generate fluid resources through simulated cards that store **fluid data**
- Need to obtain simulation data from the **Fluid Laser Base**
- Fluid data can only be obtained from recipes of **Fluid Laser Base** that **do not require mob**

##### Compact with Ars Nouveau
- The recipes for the **Fluid Laser Base** that **requires mob** can now be replaced with **the mob jar contains corresponding mob**(Need to be placed under the machine)
- The required warden for the **Soul Laser Base** can be the replaced with **the mob jar contains warden**(Need to be placed under the machine)
